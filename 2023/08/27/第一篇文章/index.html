html格式

meta

定义其他标签不能定义的东西 写在里面，字符编码标准。 name 指定数据的名称 content 指定数据内容 description 网站的描述 重定向到另一个网页：

转义字符

&nbsp 空格 &gt 大于号 &lt 小于号

块元素和行内元素

块元素：网页通过块元素进行页面布局 但p元素内 不能放块 行内元素：包裹文字

结构化语义标签

头部
主体
底部 nav 网页导航 Aside 侧边栏 Article 独立文章 Section 独立区块

常用： div 没有语义 表示一个区块 可以代替上面的 span 行内元素 选中文字

列表

可嵌套

超链接

行内元素，但是可以嵌套除它自身外的任何元素

Target

id属性

——每个标签都可添加id ——唯一标识 不重复 区分大小写 字母开头

图片标签

`图片标签向当前页面中引入一个外部图片，用img标签引入图片，img是自结束标签。

属于替换元素，介于块元素和行内元素之间。具有两种元素的特点 属性： +src 属性指定的的是外部图片路径,可以是网址 +alt 图片的描述 浏览器无法加载时显示。搜索图片的时候使用。` +width 宽度（单位是像素） +height 高度 只修改一个 会等比例缩放。 格式： jpeg：颜色丰富，不透明，不支持动图（照片） gif：动图 简单透明 颜色少 png:不支持动图 复杂透明 颜色多（网页） webp：谷歌推出专用网页。兼容不好。 效果一样用小的，效果用好的。 base64：将图片转换为字符，一般用于和网页一起加载的图片。

内联框架

  • src 指定要引入的网页的路径 +frameborder 边框 0无边框 1有边框`

音视频

`audio 向页面引入一个外部音频文件。默认不允许用户控制，加上一个controls用户就可以控制 属性：

  • controls 使用允许用户控制
  • autoplay 音乐在打开页面时会自动播放，但目前大部分浏览器不会自动播放音乐
  • loop 循环播放`
